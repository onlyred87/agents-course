# 퀴즈 3: 도구

1. 도구란 무엇인가?
    - a) LLM이 직접 실행하는 함수
    - b) LLM에 제공되는 외부 함수
    - c) 시스템 프롬프트의 일부
    - d) LLM의 파라미터

2. 도구를 LLM에 제공하는 가장 일반적인 방법은?
    - a) 함수 코드를 직접 삽입
    - b) 시스템 프롬프트에 도구 설명 추가
    - c) LLM에 직접 실행 권한 부여
    - d) 도구를 LLM에 학습시킴

3. 도구의 입력값을 명확히 정의해야 하는 이유는?
    - a) LLM이 임의로 입력값을 생성하지 않도록
    - b) 도구 호출 오류 방지
    - c) LLM이 도구를 올바르게 사용하도록
    - d) 모두 해당

정답:
1-b, 2-b, 3-d
