# 그리고 이제? 어떤 주제를 더 배워야 할까요?

에이전트형 AI는 빠르게 발전하는 분야이며, 지능적이고 자율적인 시스템을 구축하려면 기본 프로토콜에 대한 이해가 필수적입니다.

특히 익숙해져야 할 두 가지 중요한 표준이 있습니다:

- **모델 컨텍스트 프로토콜(Model Context Protocol, MCP)**  
- **에이전트-투-에이전트 프로토콜(Agent-to-Agent Protocol, A2A)**

## 🔌 모델 컨텍스트 프로토콜(MCP)

Anthropic의 **모델 컨텍스트 프로토콜(MCP)**은 AI 모델이 외부 도구, 데이터 소스, 애플리케이션과 **안전하고 원활하게 연결**될 수 있도록 해주는 오픈 스탠다드입니다. 이를 통해 에이전트는 더욱 강력하고 자율적으로 동작할 수 있습니다.

MCP는 각 환경마다 별도의 통합 작업 없이, AI 모델이 다양한 디지털 환경에 **범용 어댑터(USB-C 포트처럼)**로 연결될 수 있게 해줍니다.

MCP는 업계에서 빠르게 확산되고 있으며, OpenAI, Google 등 주요 기업들도 도입을 시작했습니다.

## 🤝 에이전트-투-에이전트(A2A) 프로토콜

Google은 Anthropic의 MCP와 보완적으로 작동하는 **에이전트-투-에이전트(A2A) 프로토콜**을 개발했습니다.

MCP가 에이전트를 외부 도구와 연결한다면, **A2A는 에이전트끼리 서로 연결**하여 협력적이고 다중 에이전트 시스템이 복잡한 문제를 함께 해결할 수 있도록 합니다.
