가장 흥미로운 AI 주제, **에이전트**에 오신 것을 환영합니다!

이 무료 코스는 **초보자부터 전문가까지** AI 에이전트를 이해하고, 사용하고, 구축하는 여정을 안내합니다.

이 첫 번째 유닛에서는 다음을 도와드립니다:

- **코스 커리큘럼** 확인
- **진행 경로 선택**(셀프 오딧 또는 인증 과정)
- **인증 과정 및 마감일 정보**
- 코스 팀 소개
- **Hugging Face 계정 생성**
- **디스코드 서버 가입** 및 동료/운영진과의 만남

시작해봅시다!

## 이 코스에서 무엇을 배우나요? [[expect]]

- 📖 **이론, 설계, 실습**에서 AI 에이전트 학습
- 🧑‍💻 [smolagents](https://huggingface.co/docs/smolagents/en/index), [LlamaIndex](https://www.llamaindex.ai/), [LangGraph](https://langchain-ai.github.io/langgraph/) 등 **주요 라이브러리 사용법** 학습
- 💾 Hugging Face Hub에 **에이전트 공유** 및 커뮤니티 에이전트 탐색
- 🏆 **챌린지 참가** 및 다른 학생들과 에이전트 성능 비교
- 🎓 **과제 완료 시 수료증 발급**

이 외에도 다양한 내용이 준비되어 있습니다!

이 코스가 끝나면 **에이전트의 원리와 최신 도구로 직접 구축하는 방법**을 이해하게 됩니다.

**<a href="https://bit.ly/hf-learn-agents">코스 등록</a>**도 잊지 마세요!

(개인정보는 존중합니다. 이메일은 각 유닛 공개, 챌린지/업데이트 안내에만 사용됩니다.)

## 코스 구성은 어떻게 되나요? [[course-look-like]]

- *기초 유닛*: 에이전트 **이론 개념** 학습
- *실습*: **주요 라이브러리 실습** 및 환경에서 에이전트 훈련(Hugging Face Spaces 활용)
- *실전 과제*: 실제 문제 해결에 개념 적용
- *챌린지*: 직접 만든 에이전트로 경쟁 및 [리더보드](https://huggingface.co/spaces/agents-course/Students_leaderboard)에서 성능 비교

**이 코스는 여러분의 피드백과 기여로 계속 발전합니다!** [GitHub에서 이슈/PR](https://github.com/huggingface/agents-course) 환영, 디스코드 토론도 참여해 주세요.

수강 후 [👉 피드백 제출](https://docs.google.com/forms/d/e/1FAIpQLSe9VaONn0eglax0uTwi29rIn4tM7H2sYmmybmG5jJNlE5v0xA/viewform?usp=dialog)도 가능합니다.

## 커리큘럼은 어떻게 되나요? [[syllabus]]

| 챕터 | 주제 | 설명 |
| :---- | :---- | :---- |
| 0 | 온보딩 | 도구 및 플랫폼 준비 |
| 1 | 에이전트 기초 | 도구, 사고, 행동, 관찰, 포맷, LLM, 메시지, 토큰, 템플릿, 파이썬 함수 예시 |
| 1.5 | 보너스: 함수 호출용 LLM 파인튜닝 | LoRa로 함수 호출 수행 모델 파인튜닝 |
| 2 | 프레임워크 | 주요 라이브러리(smolagents, LangGraph, LLamaIndex)에서의 구현 이해 |
| 2.5 | 보너스: 에이전트 관측/평가 | 에이전트 추적 및 평가법 학습 |
| 3 | 실전 사례 | 실제 사례 구축(경험자 PR 환영) |
| 4 | 최종 과제 | 벤치마크용 에이전트 구축 및 리더보드 도전 🚀 |

보너스 유닛:
- *보너스 1*: 함수 호출용 LLM 파인튜닝
- *보너스 2*: 에이전트 관측/평가
- *보너스 3*: 포켓몬 게임 에이전트

예시: 보너스 3에서는 포켓몬 배틀용 에이전트 구축을 배웁니다 🥊.

## 인증 과정 [[certification-process]]

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit0/three-paths.jpg" alt="Two paths" width="100%"/>

*오딧 모드* 또는 *과제 수행 후 인증서* 중 선택할 수 있습니다.

오딧 모드는 모든 챌린지/과제 참여 가능, **별도 알림 불필요**.

인증 과정은 **완전 무료**:

- *기초 인증*: 1유닛 완료 시 발급(최신 트렌드 학습용)
- *수료증*: 1유닛+실전 과제+최종 챌린지 완료 시 발급

마감: **2025년 7월 1일**까지 모든 과제 제출 필요
