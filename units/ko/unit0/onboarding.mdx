### 5단계: Ollama로 로컬 모델 실행(크레딧 제한 시)

1. **Ollama 설치**
    <a href="https://ollama.com/download" target="_blank">공식 안내</a> 참고
2. **로컬 모델 다운로드**
``` bash
    ollama pull qwen2:7b # ollama 웹사이트에서 더 많은 모델 확인
```
3. **Ollama 백그라운드 실행(터미널에서)**
``` bash
    ollama serve
```
    만약 "listen tcp 127.0.0.1:11434: bind: address already in use" 오류가 발생하면 `sudo lsof -i :11434`로 해당 포트 사용 프로세스(PID) 확인. 만약 프로세스가 `ollama`라면 이미 서비스가 실행 중이니 이 단계는 건너뛰어도 됨.
4. **InferenceClientModel 대신 LiteLLMModel 사용**

   smolagents에서 `LiteLLMModel`을 사용하려면 pip로 설치
``` bash
    pip install 'smolagents[litellm]'
```
``` python
    from smolagents import LiteLLMModel

    model = LiteLLMModel(
        model_id="ollama_chat/qwen2:7b",  # Ollama 지원 모델 사용 가능
        api_base="http://127.0.0.1:11434",  # 기본 Ollama 로컬 서버
        num_ctx=8192,
    )
```
5. **작동 원리**
- Ollama는 OpenAI 호환 API(`http://localhost:11434`)로 로컬 모델 제공
- `LiteLLMModel`은 OpenAI chat/completion API 포맷 지원 모델과 통신 가능
- 즉, `InferenceClientModel` 대신 `LiteLLMModel`만 바꾸면 코드 수정 없이 사용 가능
